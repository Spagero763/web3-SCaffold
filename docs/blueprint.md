# **App Name**: Web3 Scaffold

## Core Features:

- Project Scaffold: Scaffold a Next.js 14 (App Router) project with TailwindCSS, Wagmi, and RainbowKit pre-configured.
- Wallet Connect: Enable quick wallet connect using RainbowKit on Base Mainnet.
- On-Chain Interactions: Facilitate on-chain calls using Wagmi to interact with the listed smart contracts.
- AI-Powered UI Generation: Use generative AI to automatically generate UI components based on smart contract ABIs as a tool. Describe each function, input and output to provide information to the AI to generate these components and provide appropriate testing stubs and sample calls.
- Dynamic Form Generation: Automatically render forms for interacting with smart contract functions based on the ABI definitions. Handle different input types (address, uint256, string, etc.).
- Data Display: Display fetched data from smart contracts in a structured and user-friendly format, showing balances, poll results, or note content.

## Style Guidelines:

- Primary color: A vibrant blue (#247BA0) evoking trust and technological advancement.
- Background color: Light gray (#F5F7FA), offering a clean and modern look.
- Accent color: A vivid purple (#7044ff), standing out to grab the attention to the call to action items, or important information displayed.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look for a neutral user experience.
- Code font: 'Source Code Pro' for displaying ABIs or other contract related code.
- Use modern, simple icons from a library like Remix Icon to represent different functionalities.
- A clean, grid-based layout with generous spacing to ensure readability and usability.
